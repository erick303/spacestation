Notes on the proposed dashboard: v toggles it. The title is the real APFS volume name (Macintosh HD) read from the root mount, not the generic "macOS disk". The segmented bar answers "is my disk full?" at a glance (cyan = used, green = reclaimable, ░ = free); the three stat cards below it carry the exact numbers in bordered widgets (lipgloss.RoundedBorder() + JoinHorizontal). The By Category line is one stacked bar with per-category colors, and the breakdown is capped to a single row ("+N more" instead of wrapping) — no second per-category chart, since the grouped list below is already that view. The overlay is taller than the old 3-line version but every column is fixed-width, so nothing drifts as numbers change.
